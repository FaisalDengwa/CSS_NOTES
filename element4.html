Positioning Elements in CSS

Controlling exactly WHERE an element appears on the page

CSS positioning allows you to control the exact placement of an element on a web page. There are four main positioning methods, each with its own application.

📄 All four positioning types in CSS code
/* 1. Static — default, no code needed */
div { position: static; }

/* 2. Relative — moves from its original spot */
div {
  position: relative;
  top: 20px;    /* move 20px DOWN from original */
  left: 10px;   /* move 10px RIGHT from original */
}

/* 3. Absolute — placed at exact coordinates */
div {
  position: absolute;
  top: 50px;    /* 50px from the top */
  left: 100px;  /* 100px from the left */
}

/* 4. Fixed — always visible while scrolling */
div {
  position: fixed;
  bottom: 0;     /* stuck to the bottom of the screen */
  right: 0;      /* stuck to the right of the screen */
}

Sizing Elements in CSS

Setting width and height using different methods

CSS allows you to set the width and height of an element. Proper sizing ensures elements fit well within the layout and respond to different screen sizes.

📄 Sizing examples in CSS
/* Fixed size — always 300px × 150px */
div {
  width: 300px;
  height: 150px;
  background-color: lightblue;
}

/* Percentage size — responsive to screen */
div {
  width: 50%;    /* takes half the container width */
  height: 200px;
  background-color: lightgreen;
}

/* Flexible — won't grow too big or shrink too small */
div {
  width: 80%;
  min-width: 300px;   /* never smaller than 300px */
  max-width: 600px;   /* never bigger than 600px */
}

/* Viewport — 100% of the screen, 50% of screen height */
div {
  width: 100vw;
  height: 50vh;
  background-color: peru;
}

Spacing Elements

Controlling padding and margin for clean layouts

In CSS, spacing refers to controlling the space inside and outside elements. Proper spacing ensures elements are well-separated and the content is readable. 

/* Padding — space inside the element */
div {
  padding: 20px;              /* all sides = 20px */
  background-color: limegreen;
  border: 1px solid black;
}

/* Margin — space outside the element */
div {
  margin: 30px;              /* all sides = 30px */
  background-color: white;
}

/* Centering with margin: auto */
div {
  width: 50%;
  margin: 0 auto;            /* centres horizontally */
  background-color: lightblue;
}

Full Page Layout Example — Air Tanzania Page

A two-column layout with a full-width header

This example from your textbook creates a page with a header at the top and two columns below — a left (main content) column and a right (sidebar) column. This is a very common web page layout.

📄 index.html — the HTML structure
<!DOCTYPE html>
<html>
  <head>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Two-Column Layout</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <div class="header">
      <h1>Air Tanzania</h1>
      <p>The Wings of Kilimanjaro</p>
    </div>

    <div class="container">
      <div class="left-column">
        <h2>About Us</h2>
        <p>Air Tanzania Company Limited is the flag
        carrier airline of Tanzania, based in Dar es
        Salaam, with its hub at Julius Nyerere
        International Airport.</p>
      </div>

      <div class="right-column">
        <h2>Our Fleet</h2>
        <ol>
          <li>Boeing 787-8 Dreamliner</li>
          <li>Boeing 767-300F</li>
          <li>Boeing 737 Max 8</li>
        </ol>
      </div>
    </div>
  </body>
</html>
📄 styles.css — the CSS layout code
/* Basic page styling */
body {
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}

/* Full-width header */
.header {
  background-color: #CFFFFF;
  color: #1565C0;
  padding: 20px;
  text-align: center;
  box-shadow: 0 4px 10px rgba(0,0,0,0.1);
}

/* Container for two columns */
.container {
  display: flex;        /* flexbox creates columns */
  gap: 20px;           /* space between columns */
  padding: 20px;
}

/* Left column — takes 2/3 of the space */
.left-column {
  flex: 2;
  background-color: #FFFFFF;
  padding: 20px;
  box-shadow: 0 4px 10px rgba(0,0,0,0.1);
}

/* Right column — takes 1/3 of the space */
.right-column {
  flex: 1;
  background-color: #ECEFF1;
  padding: 20px;
}

/* Responsive: stack columns on small screens */
@media (max-width: 768px) {
  .container { flex-direction: column; }
}

💡 What is display: flex?

display: flex activates Flexbox — a CSS layout system that places child elements side by side as columns. Using flex: 2 and flex: 1 means the left column gets twice the space of the right column.
