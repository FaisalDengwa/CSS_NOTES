What is the CSS Box Model?

The fundamental concept behind all CSS layout

In CSS, every HTML element is treated as a box. Even a simple paragraph or heading has a box around it. The CSS Box Model defines how this box is structured. Every box has four parts, starting from the inside and going outward.

🟡 MARGIN — space OUTSIDE the border (pushes other elements away)
🔴 BORDER — the line surrounding the element
🟢 PADDING — space INSIDE between content and border

The actual text or image

Part 1: Content

The actual text, image, or element inside the box

The content area is where your text, image, or other HTML elements appear. You control the size of the content using width and height.

div {
  width: 300px;    /* content area is 300px wide */
  height: 150px;   /* content area is 150px tall */
}
🟢

Part 2: Padding

Space INSIDE the element — between content and border

Padding adds space inside the border, between the content and the border. It makes the element bigger because the space is added inside. It also shows the element's background colour.

❌ No padding (0px)
Text is squeezed!
✅ padding: 20px
Text has breathing room
div {
  padding: 20px;            /* 20px on ALL sides */
  background-color: lightgreen;
}

/* Or set each side separately: */
div {
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 10px;
  padding-left: 20px;
}
🔴

Part 3: Border

The visible line surrounding the padding and content

The border is a line that goes around the padding and content. You can control its width (thickness), style (solid, dashed, dotted), and colour.

Border Style Examples:

Solid
border-style: solid
Dashed
border-style: dashed
Dotted
border-style: dotted
Rounded
border-radius: 20px
div {
  border: 5px solid #333;       /* width, style, colour */
}

/* Rounded corners */
div {
  border: 5px solid #333;
  border-radius: 10px;          /* rounds the corners */
}

/* Dashed red border */
div {
  border: 3px dashed red;
}
📌 The border shorthand is: width style colour

Example: border: 5px solid #333; means: 5 pixels wide, solid line, dark grey colour.

🟡

Part 4: Margin

Space OUTSIDE the element — pushes other elements away

Margin adds space outside the border, separating the element from neighbouring elements. Unlike padding, margin does NOT affect the size of the element itself.

FeaturePaddingMargin
Where is it?Inside the borderOutside the border
Does it make the element bigger?YesNo
Shows background colour?YesNo (shows page background)
PurposeBreathing room insidePushes away from neighbours
 
div {
  margin: 30px;              /* 30px gap around the element */
  background-color: white;
}

/* Centering an element horizontally */
div {
  width: 50%;
  margin: auto;              /* auto centres the element */
  background-color: lightblue;
}
💻

Full Example — Styled Border Box

All four box model parts working together
📄 HTML file (index.html)
<!DOCTYPE html>
<html>
  <head>
    <title>Border Style</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <div class="border-example">
      <h2>Styled Border Example</h2>
      <p>This box has a styled border and rounded corners.</p>
    </div>
  </body>
</html>
📄 CSS file (styles.css)
.border-example {
  border: 5px solid #333;       /* 5px wide, solid, dark grey */
  border-radius: 10px;           /* rounded corners */
  padding: 20px;                 /* space inside */
  margin: 30px auto;             /* 30px top/bottom, centred */
  background-color: lightblue;
  text-align: center;
  max-width: 400px;
}
