CSS for Graphics

CSS is not just for text — it can add images and draw shapes too!

CSS is used not only for styling text and layouts, but also for adding and manipulating graphics on web pages. It can be used to add a background image to a web page or any HTML element, add gradients to elements, and draw shapes and other graphical elements.

📌 What you will learn in this element
  • How to add a background image to a web page using CSS
  • How to create smooth colour gradients (without any image)
  • How to draw shapes (triangle, circle, sphere) using CSS only
  • How to use border styles to decorate elements
🌄

Background Images

Adding a photo or image as the background of an element

CSS can add a background image to a web page or any element. The image file must be saved in the same folder as your HTML file. Then you use the background-image property to apply it.

🖼️
background-image
Sets the image file to use as background
🔍
background-size
cover = fills entire area
contain = fits inside
🎯
background-position
Where to place the image (center, top, left, etc.)
🔁
background-repeat
no-repeat = show once only
repeat = tile the image
📄 Step 1: Create index.html
<!-- index.html -->
<!DOCTYPE html>
<html>
  <head>
    <title>Background Image</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <div class="background-example">
      <h2>The Flag of Tanzania as Background</h2>
      <p>This is a background image example using CSS.</p>
    </div>
  </body>
</html>
📄 Step 2: Create styles.css (save in the SAME folder as index.html)
.background-example {
  /* Replace 'tanzania_flag.png' with your actual image file name */
  background-image: url('tanzania_flag.png');

  background-size: cover;         /* image fills the whole container */
  background-position: center;     /* centres the image */
  background-repeat: no-repeat;   /* show image only once */

  padding: 50px;
  color: white;                   /* text colour must contrast the background */
  text-align: center;
}

⚠️ Important — Image File Location

Save your image (e.g. tanzania_flag.png) in the same folder as your HTML file. Then use the file name in the background-image property. If the file is in a folder called "images", use url('images/tanzania_flag.png').

CSS Gradients

Smooth colour transitions — no image file needed!

A gradient creates a smooth transition between two or more colours without needing an image file. There are two main types: linear gradients (straight line direction) and radial gradients (circle from the centre outward).

to right → blue to green
to bottom → green to yellow
135deg → red to blue
radial → yellow to red
📄 Linear gradient — colour goes in a straight line
.gradient-example {
  /* linear-gradient(direction, colour1, colour2) */
  background: linear-gradient(to right, blue, green);
  /* "to right" means the colour changes from LEFT (blue) to RIGHT (green) */
  padding: 50px;
  color: white;
  text-align: center;
}
📄 HTML to use the gradient
<div class="gradient-example">
  <h2>Linear Gradient</h2>
  <p>This background transitions smoothly from blue to green.</p>
</div>
📄 More gradient direction options
/* Directions you can use */
background: linear-gradient(to right, blue, green);    /* ← left to right → */
background: linear-gradient(to bottom, blue, green);   /* ↑ top to bottom ↓ */
background: linear-gradient(to left, blue, green);    /* → right to left ← */
background: linear-gradient(135deg, blue, green);      /* diagonal angle */

/* Three colours */
background: linear-gradient(to right, red, yellow, green);

Drawing Shapes Using CSS

Triangle, circle, and sphere — no drawing app needed!

You can draw shapes like a triangle, circle, and sphere using only CSS — no image files and no drawing software needed!

Triangle
Circle
Sphere

A. Circle — border-radius: 50%

A circle is created by making a square element and setting border-radius: 50%. This rounds all four corners fully into a circle.

.circle {
  width: 100px;
  height: 100px;           /* width MUST equal height for a perfect circle */
  background-color: #00F;  /* blue colour */
  border-radius: 50%;      /* this makes it circular! */
  margin: 20px auto;       /* centres it on the page */
}

B. Triangle — The Border Trick

A triangle is made by setting the width and height to 0 and using the CSS border property creatively. Three of the four borders are made transparent (invisible) and one has a colour.

How the triangle trick works:

When width and height are 0, the borders meet in the middle like a star. By making the left and right borders transparent (invisible) and giving the bottom border a colour, the coloured triangle is revealed. 

.triangle {
  width: 0;                              /* zero width! */
  height: 0;                             /* zero height! */
  border-left: 50px solid transparent;   /* invisible left side */
  border-right: 50px solid transparent;  /* invisible right side */
  border-bottom: 100px solid #f00;       /* RED visible bottom = triangle! */
  margin: 20px auto;
}

C. Sphere — Circle + Radial Gradient

A sphere is a circle with a radial gradient applied to make it look 3D. The gradient goes from white (bright highlight) in one corner to dark grey on the opposite side.

.sphere {
  width: 200px;
  height: 200px;
  border-radius: 50%;     /* makes it a circle */

  /* radial-gradient creates a 3D light effect */
  /* The light comes from the top-left corner (30% 30%) */
  background: radial-gradient(
    circle at 30% 30%,
    #ffffff,       /* bright white highlight */
    #dddddd 40%,   /* light grey */
    #999999 70%,   /* medium grey */
    #000000        /* dark shadow */
  );

  margin: 20px auto;
}

Full Example — All Three Shapes Together

📄 index.html
<!-- index.html -->
<!DOCTYPE html>
<html>
  <head>
    <title>Shapes</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <div class="container">
      <div class="column">
        <div class="triangle"></div>
        <div class="circle"></div>
      </div>
      <div class="column">
        <div class="sphere"></div>
      </div>
    </div>
  </body>
</html>
📄 styles.css (Listing 1.5 from your textbook)
/* styles.css */

.container {
  display: flex;
  justify-content: space-around;
  height: 100vh;    /* full viewport height */
}

.column {
  display: flex;
  flex-direction: column;
  align-items: center;
  justify-content: center;
  gap: 20px;
}

/* Triangle */
.triangle {
  width: 0;
  height: 0;
  border-left: 50px solid transparent;
  border-right: 50px solid transparent;
  border-bottom: 100px solid #f00;
  margin: 20px auto;
}

/* Circle */
.circle {
  width: 100px;
  height: 100px;
  background-color: #00F;
  border-radius: 50%;
  margin: 20px auto;
}

/* Sphere */
.sphere {
  width: 200px;
  height: 200px;
  border-radius: 50%;
  background: radial-gradient(
    circle at 30% 30%,
    #ffffff, #dddddd 40%, #999999 70%, #000000
  );
  box-shadow: inset -20px -20px 30px rgba(255,255,255,0.5),
               inset 20px 20px 40px rgba(0,0,0,0.5);
  margin: 20px auto;
}
