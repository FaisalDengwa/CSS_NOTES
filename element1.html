Key CSS Terms You Must Know First

Before adding colour, understand how CSS is written

Every CSS rule has three parts: a selector, a property, and a value. Look at this diagram carefully — it shows exactly how CSS is written:

💡 Remember

CSS stands for Cascading Style Sheets. It is always connected to your HTML file using this line inside <head>:

<link rel="stylesheet" href="styles.css">

Method 1: Named Colours

The simplest way — just type the colour name in English

CSS understands many colour names in plain English. You just write the name and the browser knows exactly what colour to use.

Redred
Blueblue
Greengreen
Yellowyellow
Orangeorange
Purplepurple
Pinkpink
Blackblack
Whitewhite
Light Bluelightblue
📄 Example — Using named colours in HTML (inline style)
<p style="color: green;">This text is green.</p>
<h1 style="color: red;">This heading is red.</h1>
<p style="background-color: yellow;">Yellow background.</p>
📄 Example — Using named colours in a CSS file (styles.css)
/* This styles a heading element */
h1 {
  color: green;
  background-color: black;
}

/* This styles all paragraph elements */
p {
  color: blue;
}

Method 2: Hex Codes (Hexadecimal)

More precise colours using a 6-character code

A hex code is a 6-character code starting with #. It has three pairs of characters representing Red, Green, and Blue. Values go from 00 (none of that colour) to FF (full amount of that colour).

#
RRRed (00–FF)
GGGreen (00–FF)
BBBlue (00–FF)
ColourHex CodeWhat the code meansPreview
Pure Red#FF0000Full red, no green, no blue
Pure Green#00FF00No red, full green, no blue
Pure Blue#0000FFNo red, no green, full blue
Black#000000None of any colour
White#FFFFFFFull of every colour
Tanzania Green#1EB53AMixed red, high green, low blue
Dark Grey#333333Equal small amounts of each
✏️ Short Hex Codes — A Quick Shortcut

When both letters in each pair are the same, you can shorten to 3 characters:

  • #FF0000 → #F00 (red)
  • #333333 → #333 (dark grey)
  • #FFFFFF → #FFF (white)
📄 Example — Hex codes in CSS
h1 {
  color: #FF0000;           /* red text */
  background-color: #000000;  /* black background */
}
p {
  color: #1EB53A;           /* Tanzania green */
  background-color: #FFF;    /* short hex for white */
}

Method 3: RGB Values

Mix Red, Green, Blue with numbers from 0 to 255

RGB means Red, Green, Blue. You mix these three colours by giving each a number from 0 (none) to 255 (maximum).

📄 Example — RGB colours in CSS
p  { color: rgb(0, 0, 255); }      /* blue text */
h1 { background-color: rgb(255, 165, 0); } /* orange background */

Full Working Example — Two Files

Create these two files and open index.html in your browser
1

Create index.html

This is the HTML file with your web page content.

<!-- index.html -->
<!DOCTYPE html>
<html>
  <head>
    <title>Colour Example</title>
    <link rel="stylesheet" href="styles.css">
  </head>
  <body>
    <h1>Welcome to Tanzania</h1>
    <p>Tanzania is a beautiful country.</p>
    <p class="highlight">Mount Kilimanjaro is the highest peak in Africa.</p>
  </body>
</html>
2

Create styles.css

This file controls all the colours. Save it in the SAME folder as index.html.

/* styles.css */

body {
  background-color: #F0FFF0;   /* very light green background */
}

h1 {
  color: green;                 /* named colour */
  background-color: #000000;   /* hex code for black */
  padding: 10px;
}

p {
  color: #1565C0;               /* dark blue using hex */
}

.highlight {
  background-color: rgb(255, 245, 157);  /* light yellow using RGB */
  color: #000;
  padding: 8px;
}
3

What you will see

Open index.html in a browser. The heading will be green text on black, and the last paragraph will have a yellow background.
