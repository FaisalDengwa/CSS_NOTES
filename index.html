📘 What is CSS?

  • CSS stands for Cascading Style Sheets.
  • CSS is used to make HTML web pages look beautiful.
  • Without CSS, a web page looks plain and boring. With CSS, you can change colours, sizes, fonts, and layouts.
  • CSS is written using a simple pattern: selector { property: value; }
  • Example: p { color: blue; } — this makes all paragraph text blue.
